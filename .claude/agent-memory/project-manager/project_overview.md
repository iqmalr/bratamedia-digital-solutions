Bratamedia Digital Solutions is a software house landing page and portfolio website (Phase 1), evolving into a fullstack app with admin panel and blog/article CMS (Phase 2).

**Why:** Client needs an online presence and portfolio showcase before the fullstack platform is built.
**How to apply:** Keep all Phase 1 tasks focused on static/SSR landing page delivery. Do not scope Phase 2 work into Phase 1 sprints without explicit approval.

## Tech Stack
- Next.js 16 (App Router), TypeScript strict, React 19, Tailwind CSS v4, shadcn/ui, Framer Motion
- i18n: next-intl, bilingual ID/EN
- Deployment: Vercel

## Landing Page Sections
1. Hero — main banner with CTA
2. Services — software house service offerings
3. Portfolio — showcase of past projects
4. Testimonials — client reviews
5. Contact — contact form with Server Action
