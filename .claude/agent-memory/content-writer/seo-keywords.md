## Notes for Future Pages
- Blog articles: choose distinct long-tail keywords to avoid competing with the homepage
- Portfolio detail pages: target project-specific or technology-specific keywords (e.g., "React Native app development Indonesia")
- Do NOT use "software house Indonesia" as primary keyword on any page other than the homepage — it is already claimed there

**Why:** Keyword cannibalization would hurt homepage ranking for the most valuable commercial intent keyword.

**How to apply:** Before writing SEO metadata for any new page or article, check this file first. If the desired keyword is already used, choose a longer-tail variant.
