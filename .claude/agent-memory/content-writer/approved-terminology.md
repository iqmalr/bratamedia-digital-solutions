## Translation Decisions

| English Term | Approved Indonesian | Notes |
|---|---|---|
| Web Development | Pengembangan Web | Standard; do not use "Pembuatan Website" in section headings |
| Mobile App | Aplikasi Mobile | Preferred over "Aplikasi Ponsel"; industry standard |
| Custom Software | Perangkat Lunak Kustom | Keep "Kustom" — "khusus" works too but "kustom" is more industry-idiomatic |
| Cloud & DevOps | Cloud & DevOps | Keep in English — both terms are universally used in Indonesian tech industry |
| UI/UX Design | Desain UI/UX | Keep acronym; spell out in body copy as "antarmuka" (interface) and "pengalaman pengguna" |
| SaaS | SaaS | Keep in English — fully absorbed into Indonesian tech vocabulary |
| MVP | MVP | Keep in English — universally understood in startup contexts |
| E-Commerce | E-Commerce | Keep hyphenated English form |
| Tech Stack | Stack teknologi / teknologi | Use "teknologi terkini" in marketing copy; "stack" acceptable in technical contexts |
| Startup | Startup | Keep in English — standard in Indonesian business vocabulary |
| Deploy / Deployment | Deploy / Deployment | Keep in English — no natural Indonesian equivalent in common use |

## Address Format (Indonesian)
`Jl. [Street Name] No. [Number], [District], [City], [Province] [PostalCode]`
Example: `Jl. Inovasi Digital No. 1, Jakarta Selatan, DKI Jakarta 12345`

## Phone Format (Indonesia)
- Local landline: `+62 21 XXXX XXXX`
- Mobile: `+62 812 XXXX XXXX`
- Country code +62 always shown in business contexts
